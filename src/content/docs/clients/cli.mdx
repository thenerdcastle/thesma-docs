## Install

The CLI ships with the Python SDK — no separate install.

```bash
pip install thesma
thesma --help
```

## Auth

The CLI reads `THESMA_API_KEY` from the environment:

```bash
export THESMA_API_KEY="gd_live_..."
thesma companies list --ticker AAPL
```

No `--key` flag — use the env var.
