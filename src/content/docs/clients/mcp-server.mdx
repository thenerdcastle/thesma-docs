## Two ways to use

1. **Hosted** — connect `mcp.thesma.dev` as an integration in Claude Desktop, Claude.ai, or Cursor. OAuth flow signs you in with your Thesma account. No key-wrangling — your API key **is** the OAuth access token.
2. **Self-host** — `pip install thesma-mcp` and run locally with your own API key.

## Tool surface

40+ tools across:

- **SEC** — companies, filings, financials, ratios, screener, events, insider trades, institutional holdings, compensation, board members
- **Census** — places, metrics, breakdowns
- **BLS** — industries, employment, wages, occupations, turnover, unemployment
- **SBA** — lending by county/state/industry, lender detail
- **Funds** — holdings, holding changes

Ask your AI assistant a real business question ("what's Apple's 5-year revenue trend vs. their industry employment trend?") and it'll compose the right tool calls and return structured data.
